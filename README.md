<h1>Remixml-forms</h1>

Remixml-forms is a form generator using Remixml.

## Requirements

It runs inside any webbrowser environment (starting at IE11 and up).

## Basic usage

```js
 {
 {_: {
    sitename: "foo.bar",
    description: "faster than lightning templates"
  },
    whatever: 7
  }
 });
```
